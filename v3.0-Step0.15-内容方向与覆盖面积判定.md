# v3.0 Step 0.15: 内容方向与覆盖面积判定

> **插入位置**：Step 0.1（素材清点）和 Step 0.2（素材质量评级）之间
> **目的**：在清点完素材之后、评级之前，先回答三个问题——内容方向是什么？覆盖面积是什么？实际素材支撑了什么？
> **来源**：用户输入的核心理念——"第一步决定了后续的高度和深度"

---

## 为什么这一步必须存在

现有 Step 0.1 回答了"有什么素材"，Step 0.2 评级了"素材质量如何"，但跳过了最关键的一步：

**素材本身不能告诉你该往哪个方向蒸馏。**

同样45分钟的对话素材：
- 如果是连麦诊断 → 蒸馏方向是"诊断引擎+问题类型学"
- 如果是创作心得分享 → 蒸馏方向是"创作方法论+审美体系"
- 如果是商业判断讨论 → 蒸馏方向是"决策框架+风险评估"

**素材类型相同，但内容方向不同 → 蒸馏策略完全不同。**

---

## 提示词

```
你是一个人物蒸馏系统的内容方向判定模块。你的任务是：在清点完素材之后，回答三个问题——此人的内容方向是什么？覆盖面积是什么？实际素材能支撑什么？

## 输入
- Step 0.1: 素材清点结果（类型、数量、时间范围）
- 素材内容样本（关键对话/文章片段的主题分布）

## 三个核心问题

### 问题1: 内容方向是什么？
此人的内容输出主要在谈论什么？按主题聚类，用占比说话。

**不是问"他是什么职业"——是问"他说的话主要在讲什么"。**
- 鲁大魔的职业是"自媒体人"，但他90%的内容方向是"诊断别人的商业问题"
- Naval的职业是"投资人"，但他的内容方向是"人生哲学+投资判断"各占一半

### 问题2: 覆盖的面积是什么？
此人的内容覆盖了哪些主题领域？每个领域的深度如何？有没有明显的"他应该谈但没谈"的领域？

这是为了后续判断：蒸馏产物应该覆盖哪些领域？诚实边界应该标注哪些领域？

### 问题3: 实际素材能支撑什么？
基于现有素材，哪些蒸馏方向是可执行的？哪些是素材不足以支撑的？

**这是最关键的问题——它直接决定了后续Phase 1-3的维度激活策略。**

## 执行步骤

### Step 1: 内容方向判定

扫描所有素材的标题/话题/关键词，按主题聚类。用实际出现频率说话，不用猜测。

| 主题领域 | 素材占比 | 具体体现 | 判断依据 |
|---------|---------|---------|---------|
| [领域A] | X% | [此人主要在谈什么] | N条素材覆盖此主题 |
| [领域B] | Y% | ... | ... |
| ... | ... | ... | ... |

**规则**：
- 占比≥30% → 核心方向（后续蒸馏必须覆盖）
- 占比10-30% → 次要方向（选2-3个最有特色的做）
- 占比<10% → 边缘方向（如果特有维度触发则做，否则不进诚实边界）

### Step 2: 覆盖面积分析

画出此人的内容覆盖地图：

```markdown
## 内容覆盖地图 — [人物名]

### 已覆盖的领域
| 领域 | 深度 | 广度 | 此人的独特视角 |
|------|------|------|-------------|
| [领域A] | 深/中/浅 | 广/中/窄 | [此人在这个领域说了什么别人没说过的] |
| ... | ... | ... | ... |

### 明显缺失的领域
[此人基于他的类型/身份，应该谈论但未谈论的领域]
- 例如：一个诊断型人物但从未谈过"如何处理自己的失败"
- 例如：一个产品型人物但从未谈过"什么时候该砍掉一个产品"

### 内容的时间分布
| 时间段 | 主要内容方向 | 是否有方向变化 |
|--------|------------|-------------|
| YYYY-YYYY | ... | ... |
```

### Step 3: 素材支撑度评估

逐项评估每个蒸馏方向是否有足够的素材支撑：

| 蒸馏方向 | 所需素材类型 | 现有素材 | 支撑度 | 可否执行 |
|---------|------------|---------|--------|---------|
| [方向A] | 对话+著作 | 12段对话+3篇长文 | 🟢 充足 | ✅ |
| [方向B] | 决策记录 | 仅1次提及 | 🔴 不足 | ❌ → 进入诚实边界 |
| ... | ... | ... | ... | ... |

### Step 4: 蒸馏策略生成

基于以上三步，生成此人的蒸馏策略：

```markdown
## 蒸馏策略 — [人物名]

### 核心蒸馏方向（必须做，素材充足）
1. [方向A] — 覆盖领域: [X] — 素材支撑度: 🟢
2. [方向B] — 覆盖领域: [Y] — 素材支撑度: 🟢

### 补充蒸馏方向（素材不足但值得标注）
1. [方向C] — 现有素材: [X条] — 风险: [什么结论可能不可靠]

### 不做的方向（明确标注原因）
1. [方向D] — 原因: [素材不足/不在此人核心活动中/与蒸馏目标无关]

### 此人的"独特性"预判
基于内容方向分析，此人的蒸馏产物最独特的价值可能在于：
- [预判1]
- [预判2]

### 诚实边界预补充
- 缺失领域 [X] → 蒸馏产物不应该声称覆盖此领域
- 素材不足以支撑 [方向Y] → 如果用户问到，必须诚实标注
```

## 下游依赖
→ Step 0.2 素材质量评级（质量评级结合方向判定，才能判断"素材够不够做这个方向"）
→ Step 0.3 人物类型判定（内容方向是类型判定的核心输入——看他说什么来判断他是什么）
→ Step 0.4 维度激活清单（内容方向直接决定哪些维度激活、哪些降级）

## 质量检查
- [ ] 三个问题全部回答（方向/面积/支撑度）
- [ ] 内容方向有具体的素材占比（不是感觉）
- [ ] 缺失领域已标注（不是"可能还有"）
- [ ] 蒸馏策略给出了明确的核心/补充/不做三个档次
- [ ] 诚实边界预补充了因为素材不足而不能做的方向
```

## 实际案例：用鲁大魔验证此步骤

以鲁大魔为例，验证 Step 0.15 的产出：

### 1. 内容方向
| 主题领域 | 素材占比 | 核心/次要/边缘 |
|---------|---------|-------------|
| 商业诊断（连麦解决问题） | 70% | 核心 |
| 商业思维教学 | 15% | 次要 |
| 个人经历/故事 | 10% | 边缘 |
| 纯哲学/抽象讨论 | 5% | 边缘 |

→ 核心蒸馏方向：诊断引擎。次要：教学方法论。边缘：个人叙事风格。

### 2. 覆盖面积
- 已覆盖：商业判断（深）、营销获客（深）、个人定位（中）、团队管理（浅）
- 明显缺失：**此人几乎不谈"自己失败过的案例"**（只有零星提及）→ 诚实边界
- 明显缺失：**此人几乎不谈"财务/具体的数字"** → 投资型维度不激活

### 3. 素材支撑度
| 蒸馏方向 | 支撑度 |
|---------|--------|
| 问题类型学（诊断引擎） | 🟢 904条对话中约700条是诊断 |
| 思维武器（操作方法） | 🟢 17种武器有频率统计 |
| 个人决策记录 | 🔴 不足 → 不做 |
| 审美体系 | 🔴 不足 → 不做（他不是创作型） |

### 4. 蒸馏策略
- 核心方向：诊断引擎 + 思维武器 + 问题类型学
- 不做方向：决策日志分析（素材不足）、审美体系（不在此人核心活动）
- 独特性预判：此人的"破锁"操作可能是整个框架中最独特的维度

## 一句话总结

**Step 0.15 回答了你说的三个问题——方向、面积、素材——在动手蒸馏之前，先搞清楚"往哪挖"和"挖不挖得动"。没有这一步的蒸馏=蒙眼挖矿。**
