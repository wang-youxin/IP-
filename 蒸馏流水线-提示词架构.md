## Step 0.4: 维度激活清单生成（运行红色判断规则）

### 提示词

```
你是一个人物蒸馏系统的规则引擎。综合 Phase 0 前三步的全部结果，运行8条红色判断规则，输出此人的完整维度激活清单。

- Step 0.1: 素材清点结果
- Step 0.2: 素材质量评级
- Step 0.3: 人物类型判定

## 红色判断规则（逐条运行）

### 规则R1: 素材类型 → 激活特有维度
逐条检查素材类型，触发对应维度：

| 素材中存在 | 激活维度 | 原因 |
|-----------|---------|------|
| 直播/对话/连麦素材 | 即兴思维模式分析 | 对话中的反应模式≠著作中的系统思考 |
| 本人著作/长文 | 系统思考框架 | 长文揭示反复出现的核心论点（≥3次=真信念） |
| 社交媒体(Twitter/微博等) | 碎片表达DNA | 短文案揭示高频词、争议立场、幽默方式 |
| 内部文档/备忘录 | 私下判断vs公开表达对比 | 内部文档可能暴露公开表达中被隐藏的真实判断 |
| 视频/音频素材 | 语音特征分析 | 语速、停顿、音量变化揭示确定性/犹豫模式 |
| 决策记录/投资记录 | 决策日志分析 | 实际决策vs声称原则的对比 |
| 代码/设计稿/作品 | 作品分析 | 作品=最诚实的思维证据 |

### 规则R2: 人物类型 → 激活插件（作为探测起点，非执行模板）⚠️v3.2
根据主类型+次类型，激活对应插件作为**探测起点**（详见 Part 5）。插件不是执行模板——素材可以推翻插件预设：

> ⚠️ **v3.2升级**: 插件维度是"探测起点"而非"必做清单"。当素材显示此人不用该类型的常规方法时（R7触发），插件维度被标记为"此人偏离"，偏离本身成为分析对象。

| 主类型 | 激活插件(探测起点) | 核心维度（可被素材推翻） |
|--------|-------------------|---------------------|
| 诊断型 | 诊断插件 | 问题类型学、诊断引擎、武器组合表 |
| 创作型 | 创作插件 | 创作方法论、审美体系、风格演进、作品谱系 |
| 产品型 | 产品插件 | 产品哲学、设计原则、UX方法论、迭代逻辑 |
| 学术型 | 学术插件 | 研究方法论、理论体系、实验设计、学术谱系、教学法 + 🆕行为模式异常检测 |
| 投资型 | 投资插件 | 决策框架、风险评估、持仓管理、市场心理 |
| 哲学型 | 哲学插件 | 论证结构、核心命题、逻辑链、思想实验 |
| 表演型 | 表演插件 | 舞台方法论、观众心理、节奏控制、即兴规则 |
| 管理型 | 管理插件 | 组织哲学、决策框架、人才体系、危机处理 |

**插件推翻协议**：
1. 如果Phase 2发现此人在插件预设维度上"无行为"（如学术型但不写论文）→ 不强行分析"他为什么不用"，改为分析"他用什么替代了"
2. 如果Phase 2发现此人的行为与插件预设维度矛盾 → 标注"此人偏离[类型]常规"，偏离本身成为发现
3. 次类型 → 激活补充维度（从对应插件中选取2-3个最相关的作为探测起点）

### 规则R3: 素材质量 → 深度边界
- 一手>70% + 时间>10年 → 可做"心智模型三重验证"
- 一手40-70% → 心智模型改为"观点聚类"，不声称是"模型"
- 一手<40% → 所有结论标注"基于二手资料推测"

### 规则R4: 时间跨度 → 思想演进深度
- >10年 → 激活"思想演进图"（标注关键转折点+立场变化）
- 3-10年 → 激活"关键转折点"（不画完整演进图）
- <3年 → 标注"快照"，诚实边界注明"未覆盖思想演进"

### 规则R5: 信息缺口 → 诚实边界补充
对于以下缺失，自动进入诚实边界：
- 无著作/长文 → 诚实边界: "缺少系统性书写，心智模型仅从对话/碎片推断"
- 无对话/即兴素材 → 诚实边界: "仅有精心编辑的内容，即兴思维未知"
- 无他者视角 → 诚实边界: "缺少外部批评交叉验证"
- 无时间线 → 诚实边界: "思想演进未覆盖"
- 无决策记录 → 诚实边界: "实际决策vs声称原则无法对比"

### 规则R6: 内容方向 → 维度权重调整 🆕v3.2
Step 0.15的内容方向聚类结果不是仅供参考——它直接调整后续所有维度的权重：
- 核心方向(≥30%) → 该方向的维度权重×2，必须深度执行
- 次要方向(10-30%) → 标准权重，选取最有特色的2-3个
- 边缘方向(<10%) → 权重×0.5，仅当涌现模式触及时才做
- 职业标签≠内容方向时 → 以内容方向为准，职业类型插件降为参考

**示例**：费曼的职业=物理学家(学术型)，但80%内容方向=科学思维方式+教学与理解。
→ R6判定：调高"思维表征方式"和"教学法"权重，调低"理论体系"和"实验设计"权重。
→ 学术型插件A2(理论体系)和A3(实验设计)降为参考，不做深度执行。

### 规则R7: 行为模式异常 → 特有维度触发 🆕v3.2
当素材显示此人**不按其所属类型的常规方式做事**时，触发"反常规行为模式"维度：
- 学术型但不读文献/不写论文 → 触发"学术非标模式"
- 投资型但不看财报/不分析基本面 → 触发"投资非标模式"  
- 产品型但不用用户研究/不写PRD → 触发"产品非标模式"
- 任何类型的"反常规"达到≥3次 → 触发"行为模式异常分析"

**核心逻辑**：此人不用常规方法却能出成果——说明他有替代机制。这个替代机制往往是他最独特的地方。

**执行位置**：Phase 2 Step 2.0 扫描时自动标记"反常规行为"，Step 2.1 聚类时作为独立聚类分析。

### 规则R8: 表达与思维关系判定 🆕v3.2
判断此人的表达方式是"修辞工具"还是"思维本身的一部分"：
- **修辞工具型**：表达服务于沟通——此人先想清楚，再用表达技巧传递。表达DNA和概念体系可分拆分析。
- **思维一体型**：表达即思维——此人在表达的过程中完成思考。表达DNA和概念体系不可分拆。

**判定信号**：
| 信号 | 修辞工具型 | 思维一体型 |
|------|----------|----------|
| 即兴vs准备 | 准备稿和即兴表达风格相似 | 即兴表达比自己写的好/不同 |
| 抽象→具象 | 先有抽象结论，再加例子 | 从具体例子跳跃到抽象，无中间层 |
| 写作方式 | 先列大纲，再填充 | 边写边想，写完才知道结论 |
| 自述 | "我只是把想好的说出来" | "我不知道我怎么想的，直到我说出来" |

**激活条件**：当素材同时满足①有即兴+准备两种素材类型 ②表达DNA显示即兴和准备风格显著不同 ③此人谈论过自己的思维过程 → 激活"思维-表达一体性分析"

**执行位置**：Phase 4 交叉分析——表达DNA × 智识谱系 × 时间线 三重交叉中增加此判定。

**🆕 v3.8 R8后续行动**（修复C1：判断完之后必须有实际行动）：

一旦R8判定完成，根据结果执行以下不同的下游策略：

#### 如果是"修辞工具型"：
| 后续行动 | 具体操作 | 执行位置 |
|---------|---------|---------|

> ⚠️ **v3.9 修复C2**：表达DNA(Step 1.1)和思维表征方式(Step 2.0)分析的是同一件事的两个不同层面。不是重复劳动——是互补分析。

**表达DNA vs 思维表征方式 — 明确分界**：

| | 表达DNA (Step 1.1) | 思维表征方式 (Step 2.0) |
|---|-------------------|---------------------|
| **分析层级** | 表面层 — 怎么说 | 认知层 — 怎么想→怎么说 |
| **核心问题** | "此人用什么语言工具表达？" | "此人怎么把抽象思维转化为可沟通的形式？" |
| **分析对象** | 句式/词汇/节奏/语气/修辞频率 | 类比映射方向/可视化依赖/公式化倾向/故事化模式 |
| **重复分析同类现象时** | 如"喜欢用类比"→统计类比出现频率+语气特征 | 如"喜欢用类比"→分析类比来源域→目标域的映射规律+这反映了什么认知结构 |

| **重叠区（类比/故事）** | 统计层面：频率、风格、上下文 | 结构层面：映射规律、认知功能、概念依赖 |

## Step 3.5: 对抗验证（v3.3新增——独立质疑者挑战推导链）

> **插入位置**：Phase 3 对照定位完成后，Phase 4 交叉分析之前
> **目的**：解决"LLM自我验证循环"问题——同一个模型既写结论又检查自己的溯源 = 自己给自己改卷子。对抗验证用一个独立角色、不同prompt、相反目标来挑战Phase 2的方法论形式化。
> **核心原则**：质疑者不知道分析者的完整推理过程——只看结论和证据，不看分析者怎么推出来的。

### 提示词

```
你是一个方法论验证系统的独立质疑者。你的唯一任务：找出Phase 2方法论形式化中的薄弱环节。你不是分析者。你不应该相信任何没有足够独立证据支撑的结论。

## ⚠️ 你的角色设定
- 你是**对抗者**，不是合作者。你的价值在于找出问题，不是确认正确。
- 默认立场：每个方法论在被充分证明之前都是可疑的。
- 你对"证据"的标准高于分析者——你要求独立验证，不接受循环推理。

## 输入
- Phase 2 全部形式化后的涌现模式（频率+案例+反例+推导链+可证伪条件+深度评级）
- Phase 3 对照结果（鲁大魔库对照+参考人物库对照+深度评级确认）
- Phase 1 通用维度报告（用于交叉检查一致性）
- 原始素材（用于直接验证关键证据）

## 5维度压力测试

### 压力测试1: 替代解释排除
对每个模式的推导链"如果X→因为Y→所以Z"：
- 有没有其他解释Y'能同样好地解释"为什么X→Z"？
- 分析者是否只排除了弱替代解释，而忽略了更强的候选？
- 如果Y和Y'都成立，此人实际遵循的是哪个？

排除标准:
- ✅ 充分排除: 替代解释被独立证据明确反驳
- ⚠️ 部分排除: 替代解释被分析者声称反驳，但反驳证据不独立（与推导链使用同一段素材）
- ❌ 未排除: 替代解释成立且分析者未处理

### 压力测试2: 反例压力测试
对每个模式的反例分析：
- 反例是否真的是反例？还是分析者把"不同场景的正常变异"误判为反例？
- 有没有素材中更明显的反例被遗漏了？
- 反例的"失效条件"解释是否太宽松？（如果什么都能用失效条件解释，就没有真正的反例了）
- 在≥3个与触发条件相似的场景中随机抽查：此人是否真的用了此模式？

### 压力测试3: 推导链断裂点
对推导链的每个环节：
- Y（核心信念）是否有独立于Z的证据？还是只从Z反推Y？
  - 🔴 如果Y的唯一证据是"他做了Z所以应该相信Y"→ 循环推理，推导链断裂
  - 🟡 如果Y有独立证据但来源相同（同一段对话/同一篇文章）→ 弱独立
  - 🟢 如果Y有跨素材类型的独立证据（对话+著作+他人观察）→ 强独立
- 推导链的"因为Y"和"所以Z"之间，有没有被跳过的中间步骤？

### 压力测试4: 频率幻觉检测
对每个声称≥60%频率的模式：
- 频率计算的分母是什么？是否包含了"此人根本没有机会使用此模式"的场景？
  - 例: 声称"85%的情况使用破锁"——但如果只在"被提问"的场景下算分母，而非全部对话场景，频率被高估
- 素材的时间分布是否均匀？如果≥80%的证据来自同一时期（如某3个月），可能是阶段性行为而非稳定模式
- 素材的类型分布是否均匀？如果只在对话中出现、著作中从未出现→可能是对话特有而非通用模式

### 压力测试5: 命名偏差检测
- 模式命名是否暗示了一个不存在的因果结构？
  - 例: "破锁"这个名字暗示"锁住了→打破它"——但如果此人只是"不进入对方框架"而没有主动"打破"，命名夸大了
- 名字是否来自此人自己的术语？如果是，他使用这个术语的上下文是否一致？
  - 此人在不同时间用这个词时，指的是同一件事吗？还是分析者把一个演变的概念固定在了早期含义上？
- 如果换成描述性命名（"当X时做Y"），这个模式还有没有方法论感？

- 哪些操作是此人的"独门"（依赖自创概念）？

- 哪个概念/观点被引用最多？（网络中心）
- 哪些概念/观点是孤立的？（可能需要更多连接或删除）
